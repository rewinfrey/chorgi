About Chorgi

An interactive music theory learning and exploration tool designed to help musicians visualize, understand, and practice chords, scales, and key relationships across multiple instruments.

Project Goals

Chorgi was created with the vision of making music theory more accessible and visual. Many musicians struggle to understand abstract concepts like chord progressions, key relationships, and modulation techniques. Chorgi bridges this gap by providing:

  • Visual Learning: See music theory concepts rendered on actual instruments (staff notation, piano, guitar)
  • Interactive Exploration: Experiment with different scales, chords, and relationships in real-time
  • Practical Application: Learn through hands-on practice with the chord recognition game
  • Deep Understanding: Discover how keys relate to each other and how to modulate between them

Core Features

Chord Explorer

Explore diatonic chords across different scales with real-time visualizations on three instruments:

  • Musical Staff (Treble Clef) with proper notation
  • Two-octave Piano Roll with highlighted chord tones
  • Guitar Fretboard showing all positions across 12 frets

Multi-Scale Overlay: Compare multiple scales simultaneously to see where they share notes and differ.

Key Analysis

Deep dive into key relationships and modulation paths:

  • Discover all 15+ musically related keys
  • Identify pivot chords between keys for smooth modulation
  • Quality scoring (0-10) for modulation effectiveness
  • Popularity indicators for relationship commonality

Comprehensive Guide: Built-in explanations of key relationships, distant modulation strategies, and famous song examples.

Practice Mode

Interactive chord recognition game to develop your skills:

  • Click notes on the piano to build chords
  • Real-time feedback and validation
  • Difficulty levels: Triads, 7th chords, Extensions
  • Scoring system with streaks and accuracy tracking
  • Lifetime statistics saved locally

Technology

Chorgi is built with vanilla JavaScript using ES6 modules and HTML5 Canvas for visualizations. The architecture emphasizes:

  • Pure Functions: Core music theory calculations are side-effect free
  • Event-Driven State: Centralized state management with event emission
  • Reusable Components: Modular visualizations that work across different modes
  • No Dependencies: Runs entirely in the browser with no external libraries

Future Vision

Theory Lab: Negative harmony calculator, chord inversions, voicing library, modal interchange explorer, circle of fifths

Progression Builder: Drag-and-drop chord sequencer with audio playback, save/share progressions, common progression library

Enhancements: Audio playback with Web Audio API, MIDI input/output support, additional instruments (bass, ukulele), mobile responsive design, offline PWA support
